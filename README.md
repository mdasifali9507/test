# PACKCORE Website — Run Locally

Live: https://packcore.lovable.app

## Requirements
- Node.js 20+ (https://nodejs.org)
- Bun (https://bun.sh)

## Steps
1. Open this folder in VS Code
2. The `.env` file is already included
3. In the terminal run:
   ```
   bun install
   bun run dev
   ```
4. Open http://localhost:8080

## Scripts
- `bun run dev` — start dev server
- `bun run build` — production build (output: dist/client)
- `bun run preview` — preview built site
